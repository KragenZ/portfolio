Case Study

AI Property
Assistant.

An intelligent RAG-based conversational agent built natively with LangChain, FAISS, and Gemini.

The Problem

Modern real estate portals force users to manually filter deeply nested databases. I built an AI agent to abstract all database logic, allowing users to simply type natural language queries like "Find me a 3BHK under $500K with a pool" and immediately receive hyper-relevant results.

The Architecture

  • Embeddings: Chunked property data into dense mathematical vectors using SentenceTransformers.
  • Vector Store: Stored millions of multi-dimensional vectors inside a local memory-bound FAISS index.
  • Generation: Passed top-K retrieved properties dynamically into the Gemini LLM for conversational wrapping.
